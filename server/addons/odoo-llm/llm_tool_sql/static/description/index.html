LLM Tool: SQL Query

Let your AI assistant query the Odoo database safely

Read-Only Secure Production Ready

How It Works

1. User Asks

"How many products do we have available?"

2. AI Queries

The AI calls execute_sql_query with a safe SELECT statement

3. AI Answers

"You have 42 products available in your system."

Security Features

  • SELECT only - INSERT, UPDATE, DELETE are blocked
  • Keyword blocking - DROP, ALTER, TRUNCATE, GRANT and more
  • Row limiting - Max 100 rows per query
  • SAVEPOINT isolation - Errors roll back safely
  • CTE support - WITH queries allowed
  • Type-safe results - Decimal, datetime auto-converted

LLM-Friendly Response Format

Count Query

SELECT COUNT(*) as count FROM product_template

Result: {"count": 42}

Multi-Row Query

SELECT name, list_price FROM product_template LIMIT 3

Result: {"rows": [
  {"name": "Desk", "list_price": 150.0},
  {"name": "Chair", "list_price": 75.0}
], "row_count": 2}

Requirements

Requires the LLM Tool base module (llm_tool). Compatible with any LLM provider (OpenAI, Ollama, Anthropic, Mistral).
